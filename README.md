# Livraria API - Backend

Este projeto é uma aplicação de livraria completa, permitindo o gerenciamento de livros e reviews. Possui um sistema de autenticação robusto com **Express Sessions**, suporte a **Tema Dark/Light** e fluxo de **Recuperação de Senha** via e-mail.
